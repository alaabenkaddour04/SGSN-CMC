# SGSN-CMC — Système de Gestion Scolaire Numérique

Projet de fin d'année — Filière Infrastructure Digitale, Option Cloud Computing
Centre de Formation CMC Nouaceur — 2026

## Stack Technique
- **Backend**: Node.js / Express
- **Base de données**: PostgreSQL
- **Frontend**: HTML / Tailwind CSS
- **Containerisation**: Docker + Docker Compose
- **Cloud**: Azure VM (Ubuntu 24.04)

## Lancer le projet
```bash
docker-compose up -d
```
